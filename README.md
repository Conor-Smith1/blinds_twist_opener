# blinds_twist_opener
A set of code, CAD Models, and wiring diagram that is for using an ESP32 device to open/close blinds with a stepper motor, zero it with a hall effect sensor, and uses 3D printed adapters to interface with the hexagonal twist blinds.

This is especially useful to assist in automating blinds in an apartment, since the control box can be secured with command strips or the like, so there is no required permanent alteration of anything.
